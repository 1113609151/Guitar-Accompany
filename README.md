# Guitar-Accompany

# 安装步骤

---

### 1.

```python
git clone git@github.com:1113609151/Guitar-Accompany.git
```

### 2.

```python
conda create -n gt -y
cd Guitar-Accompany
```

### 3.

```python
pip install -r requirements.txt
python main.py
```
